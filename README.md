# Ahmed Ramadan Portfolio

This is my personal portfolio website showcasing my skills, projects, and professional experience as a software engineer.

## Features

* Home section with introduction.
* About section with education and background.
* Training & Certificates section highlighting courses and achievements.
* Projects section with links to GitHub repositories.
* Contact section with social media links and CV.
* Responsive design and smooth scrolling animations using ScrollReveal.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* ScrollReveal.js for animations
* Boxicons for icons

## Installation / Viewing

Clone the repository:

```bash
git clone https://github.com/ahmedramadan14/portfolio-website.git
```

Navigate to the project folder:

```bash
cd portfolio-website
```

Open `index.html` in your web browser to view the portfolio.
